# 多语言模板

> [_shared.md](../_shared.md) · [config-parser.md](../config-parser.md#i18n)

按 YAML `i18n` 决定是否生成多语言。**默认 `false` = 仅中文**。

## `i18n` 开关

| `i18n` | 行为 |
|--------|------|
| `false`（**默认**） | **不更新** `zh-CN.ts` / `en.ts`；模板与 helper label 直接硬编码中文，不走 `t()`，不 `import useI18n` |
| `true` | 更新 `zh-CN.ts` + `en.ts` 的 `<i18nKey>` 分组；模板/label 使用 `t('i18nKey.xxx')` |

> 推导：YAML 未写 `i18n` 视为 `false`；显式 `i18n: true` 才走双语言。

## 规则（`i18n: true` 时生效）

- 分组 key：camelCase 特性名（如 `userMng`、`orderMng`）
- 中文值：自然中文
- 英文值：**仅首字母大写**（如 "User name" 非 "User Name"）
- Keys：camelCase

## 新建 vs 更新

| 场景 | 行为 |
|------|------|
| `<i18nKey>` 分组不存在 | 在 `};` 前**追加**新分组 |
| `<i18nKey>` 分组已存在 | **替换**整个分组（不跳过、不重复追加） |

## 模板

`src/locales/zh-CN.ts` — 在 `};` 前追加：

```typescript
  <i18nKey>: {
    <fieldKey>: '<中文标签>',
    <fieldKey2>: '<中文标签2>',
    add<Feature>: '新增<Feature>',
    edit<Feature>: '编辑<Feature>',
    delete<Feature>: '删除<Feature>',
    <feature>Management: '<Feature>管理',
    deleteSuccess: '删除成功',
    deleteConfirm: '确认删除该记录？',
  },
```

`src/locales/en.ts` — 在 `};` 前追加：

```typescript
  <i18nKey>: {
    <fieldKey>: '<English label>',
    <fieldKey2>: '<English label2>',
    add<Feature>: 'Add <feature>',
    edit<Feature>: 'Edit <feature>',
    delete<Feature>: 'Delete <feature>',
    <feature>Management: '<Feature> management',
    deleteSuccess: 'Delete successfully',
    deleteConfirm: 'Confirm to delete this record?',
  },
```

## form-only

**不生成** add/edit/delete、表格 action、deleteConfirm 等 CRUD 列表 key。

`src/locales/zh-CN.ts`：

```typescript
  <i18nKey>: {
    <fieldKey>: '<中文标签>',
    <feature>Title: '<配置 title>',
  },
```

`src/locales/en.ts`：

```typescript
  <i18nKey>: {
    <fieldKey>: '<English label>',
    <feature>Title: '<English title>',
  },
```

示例（`systemConfig`）：

```typescript
// zh-CN.ts
  systemConfig: {
    siteName: '站点名称',
    sessionTimeout: '登录超时(分钟)',
    maintenanceMode: '维护模式',
    remark: '备注',
    systemConfigTitle: '系统参数配置',
  },

// en.ts
  systemConfig: {
    siteName: 'Site name',
    sessionTimeout: 'Session timeout (min)',
    maintenanceMode: 'Maintenance mode',
    remark: 'Remark',
    systemConfigTitle: 'System config',
  },
```

## tabs

在 crud-module i18n 基础上，为每个 `tabs[]` 追加 Tab label key：

| tabs[].name | i18n key | 中文值来源 |
|-------------|----------|-----------|
| `list` | `tabList` | tabs[].label |
| `edit` | `tabEdit` | tabs[].label |

```typescript
// zh-CN.ts — demoFormTabs 示例
  demoFormTabs: {
    account: '用户账号',
    status: '状态',
    tabList: '查询-列表',
    tabEdit: '新增-修改',
    addDemoFormTabs: '新增',
    editDemoFormTabs: '编辑',
    deleteDemoFormTabs: '删除',
    demoFormTabsManagement: 'Demo 多 Tab 表单',
    deleteSuccess: '删除成功',
    deleteConfirm: '确认删除该记录？',
  },
```
